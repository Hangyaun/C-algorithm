# Linux命令

在 Linux 系统中，关键字（或称为命令和工具）是用户与系统交互的重要方式。以下是一些常见的 Linux 关键字及其用途分类：

### 文件和目录操作

- `ls`：列出目录内容。
- `cd`：更改当前目录。
- `pwd`：显示当前工作目录路径。
- `mkdir`：创建新目录。
- `rmdir`：删除空目录。
- `rm`：删除文件或目录。
- `cp`：复制文件或目录。
- `mv`：移动或重命名文件或目录。
- `touch`：创建空文件或更新文件的时间戳。
- `find`：搜索文件或目录。

### 文件内容操作

- `cat`：显示文件内容。
- `more`：分页显示文件内容。
- `less`：分页显示文件内容，支持向前向后翻页。
- `head`：显示文件的前几行。
- `tail`：显示文件的后几行。
- `grep`：在文件中搜索指定模式。
- `sed`：流编辑器，用于文本替换和操作。
- `awk`：文本处理工具，用于格式化和报告生成。

### 权限和所有权

- `chmod`：更改文件或目录的权限。
- `chown`：更改文件或目录的所有者。
- `chgrp`：更改文件或目录的组。

### 压缩和归档

- `tar`：创建和解压缩 tar 归档文件。
- `gzip`：压缩文件。
- `gunzip`：解压缩文件。
- `zip`：打包并压缩文件。
- `unzip`：解压缩文件。

### 系统信息

- `uname`：显示系统信息。
- `df`：显示文件系统磁盘空间使用情况。
- `du`：显示文件和目录的磁盘使用情况。
- `free`：显示系统内存使用情况。
- `top`：实时显示系统资源使用情况。
- `htop`：`top` 的增强版，提供更友好的用户界面。
- `ps`：显示当前进程信息。

### 网络操作

- `ping`：测试网络连接。
- `ifconfig`：配置网络接口。
- `ip`：显示或管理网络接口和路由。
- `netstat`：显示网络连接、路由表、接口统计信息等。
- `curl`：命令行工具，用于发送 HTTP 请求。
- `wget`：命令行工具，用于下载文件。

### 用户和组管理

- `useradd`：添加新用户。
- `usermod`：修改用户账户。
- `userdel`：删除用户账户。
- `groupadd`：添加新组。
- `groupmod`：修改组。
- `groupdel`：删除组。
- `passwd`：更改用户密码。

### 软件管理

- `apt-get`：Debian 系统及其衍生版本（如 Ubuntu）中的包管理工具。
- `yum`：Red Hat 系统及其衍生版本（如 CentOS）中的包管理工具。
- `dnf`：`yum` 的下一代包管理工具，适用于 Fedora。
- `pacman`：Arch Linux 的包管理工具。

### 进程管理

- `kill`：终止进程。
- `killall`：终止指定名称的所有进程。
- `pkill`：根据名称模式终止进程。
- `bg`：将作业置于后台运行。
- `fg`：将作业置于前台运行。
- `jobs`：显示当前用户的作业列表。

### 系统管理

- `sudo`：以超级用户权限执行命令。
- `su`：切换用户身份。
- `shutdown`：关闭系统。
- `reboot`：重启系统。
- `systemctl`：控制 systemd 系统和服务管理器。
- `journalctl`：查看 systemd 日志。

这些关键字（命令）是使用 Linux 系统时最常用的工具，通过熟练掌握这些命令，你可以更加高效地管理和操作 Linux 系统。
